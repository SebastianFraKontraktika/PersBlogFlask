# Blog app made with flask

## description

This is an application made with flask, with mariadb as a database.
The application is going to be a blog website which I  can add content to and others can read. Users will be able to make a user to be able to save content and comment on the topics posted.

## dependancies

``` 
pip install flask

pip install mysql-connector-python
 ```
